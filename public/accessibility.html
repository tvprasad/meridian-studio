WCAG 2.1 Level AA Section 508 Conformant

Target Standard

Meridian Studio targets WCAG 2.1 Level AA conformance. Section 508 of the Rehabilitation Act adopts WCAG 2.1 AA as its technical baseline. Manual verification was performed using VoiceOver on iOS Safari.

Known Limitations

  • Studio is an authenticated operator tool; public access is not provided.
  • DAG Trace panel contains complex SVG visualizations; screen-reader equivalents provided via text labels on each node.
  • Trademark disclaimer in footer is aria-hidden — legal boilerplate, not content.
  • WCAG 2.2 delta items targeted for a future sprint.

Scope

This statement covers the Meridian Studio operator interface at studio.vplsolutions.com. It does not cover the Meridian API backend or MCP server.

Technical Approach

  • React 19 + TypeScript, Tailwind CSS v4 — no component library
  • ARIA implemented directly, no abstraction layer
  • Streaming AI output: aria-live="polite" aria-atomic="false"
  • REFUSED responses: role="alert"
  • Animations suppressed via prefers-reduced-motion CSS block

Report an Issue

If you encounter an accessibility barrier in Meridian Studio, contact us at studio@vplsolutions.com. We aim to respond within 5 business days.
